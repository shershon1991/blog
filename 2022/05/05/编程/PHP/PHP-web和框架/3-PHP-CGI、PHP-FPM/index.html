CGIдёҺFastCGI

CGIзЁӢеәҸпјҡ

CGIзЁӢеәҸзҡ„жҜҸдёҖж¬ЎwebиҜ·жұӮйғҪдјҡжңүеҗҜеҠЁе’ҢйҖҖеҮәзҡ„иҝҮзЁӢпјҢд№ҹе°ұжҳҜжңҖдёәдәәиҜҹз—…зҡ„fork-and-executeжЁЎејҸпјҲжҜҸж¬ЎHTTPжңҚеҠЎеҷЁйҒҮеҲ°еҠЁжҖҒиҜ·жұӮж—¶йғҪйңҖиҰҒйҮҚж–°еҗҜеҠЁи„ҡжң¬и§ЈжһҗеҷЁжқҘи§Јжһҗphp.iniпјҢйҮҚж–°иҪҪе…Ҙе…ЁйғЁDLLжү©еұ•е№¶еҲқе§ӢеҢ–е…ЁйғЁж•°жҚ®з»“жһ„пјҢз„¶еҗҺжҠҠз»“жһңиҝ”еӣһз»ҷHTTPжңҚеҠЎеҷЁпјүпјҢеҫҲжҳҺжҳҫпјҢиҝҷж ·зҡ„жҺҘеҸЈж–№ејҸдјҡеҜјиҮҙphpзҡ„жҖ§иғҪеҫҲе·®пјҢеңЁеӨ„зҗҶй«ҳе№¶еҸ‘и®ҝй—®ж—¶пјҢеҮ д№ҺжҳҜдёҚеҸҜз”Ёзҡ„гҖӮжүҖд»ҘпјҢй«ҳе№¶еҸ‘ж—¶CGIзЁӢеәҸзҡ„жҖ§иғҪиҫғе·®гҖӮ

FastCGIзЁӢеәҸпјҡ

  1. Web ServerеҗҜеҠЁзҡ„еҗҢж—¶пјҢеҠ иҪҪFastCGIиҝӣзЁӢз®ЎзҗҶеҷЁпјҲnginxзҡ„php-fpm/IISзҡ„ISAPI/Apacheзҡ„Module)

  2. FastCGIиҝӣзЁӢз®ЎзҗҶеҷЁиҜ»еҸ–php.iniй…ҚзҪ®ж–Үд»¶пјҢеҜ№иҮӘиә«иҝӣиЎҢеҲқе§ӢеҢ–пјҢеҗҜеҠЁеӨҡдёӘCGIи§ЈйҮҠеҷЁиҝӣзЁӢпјҲPHP-CGIпјүпјҢзӯүеҫ…жқҘиҮӘWeb Serverзҡ„иҝһжҺҘгҖӮ

  3. еҪ“Web ServerжҺҘж”¶еҲ°е®ўжҲ·з«ҜиҜ·жұӮж—¶пјҢFastCGIиҝӣзЁӢз®ЎзҗҶеҷЁйҖүжӢ©е№¶иҝһжҺҘеҲ°дёҖдёӘCGIи§ЈйҮҠеҷЁиҝӣзЁӢгҖӮWeb Serverдјҡе°Ҷзӣёе…ізҺҜеўғеҸҳйҮҸе’Ңж ҮеҮҶиҫ“е…ҘеҸ‘йҖҒеҲ°FastCGIеӯҗиҝӣзЁӢеҚіPHP-CGIиҝӣиЎҢеӨ„зҗҶгҖӮ

  4. PHP-CGIе®ҢжҲҗеӨ„зҗҶеҗҺе°Ҷж•°жҚ®жҢүз…§CGIи§„е®ҡзҡ„ж јејҸиҝ”еӣһз»ҷWeb ServerпјҢз„¶еҗҺе…ій—ӯжҲ–иҖ…зӯүеҫ…дёӢдёҖж¬ЎиҜ·жұӮгҖӮ

FastCGIеҜ№иҝӣзЁӢзҡ„з®ЎзҗҶпјҡ

FastCGIдјҡе…ҲеҗҜеҠЁдёҖдёӘmasterиҝӣзЁӢпјҢи§Јжһҗй…ҚзҪ®ж–Үд»¶пјҢеҲқе§ӢеҢ–жү§иЎҢзҺҜеўғпјҢз„¶еҗҺеҶҚеҗҜеҠЁеӨҡдёӘworkerиҝӣзЁӢгҖӮеҪ“иҜ·жұӮиҝҮжқҘж—¶пјҢmasterиҝӣзЁӢдјҡдј йҖ’з»ҷдёҖдёӘworkerиҝӣзЁӢпјҢз„¶еҗҺз«ӢеҚіеҸҜд»ҘжҺҘеҸ—дёӢдёҖдёӘиҜ·жұӮгҖӮиҝҷж ·е°ұйҒҝе…ҚдәҶйҮҚеӨҚзҡ„еҠіеҠЁпјҢж•ҲзҺҮиҮӘз„¶жҸҗй«ҳгҖӮиҖҢдё”еҪ“workerиҝӣзЁӢдёҚеӨҹз”Ёж—¶пјҢmasterиҝӣзЁӢеҸҜд»Ҙж №жҚ®й…ҚзҪ®йў„е…ҲеҗҜеҠЁеҮ дёӘworkerиҝӣзЁӢзӯүзқҖпјӣеҪ“з©әй—ІworkerиҝӣзЁӢеӨӘеӨҡж—¶пјҢд№ҹдјҡеҒңжҺүдёҖдәӣпјҢиҝҷж ·е°ұжҸҗй«ҳдәҶжҖ§иғҪпјҢд№ҹиҠӮзәҰдәҶиө„жәҗгҖӮиҝҷе°ұжҳҜFastCGIеҜ№иҝӣзЁӢзҡ„з®ЎзҗҶгҖӮ

жҖ»з»“пјҡ

  1. CGI е’Ң FastCGI йғҪеҸӘжҳҜдёҖз§ҚйҖҡдҝЎеҚҸи®®и§„иҢғпјҢдёҚжҳҜдёҖдёӘе®һдҪ“пјҢдёҖиҲ¬иҜҙзҡ„CGIжҢҮзҡ„жҳҜз”Ёеҗ„з§ҚиҜӯиЁҖзј–еҶҷзҡ„иғҪе®һзҺ°иҜҘеҠҹиғҪзҡ„зЁӢеәҸгҖӮ

  2. CGI зЁӢеәҸе’ҢFastCGIзЁӢеәҸпјҢжҳҜжҢҮе®һзҺ°иҝҷдёӨдёӘеҚҸи®®зҡ„зЁӢеәҸпјҢеҸҜд»ҘжҳҜд»»дҪ•иҜӯиЁҖе®һзҺ°иҝҷдёӘеҚҸи®®зҡ„гҖӮпјҲPHP-CGI жҳҜдҪҝз”ЁPHPе®һзҺ°CGIеҚҸи®®зҡ„CGIзЁӢеәҸпјҢ PHP-FPMжҳҜдҪҝз”ЁPHPе®һзҺ°FastCGIеҚҸи®®зҡ„FastCGIзЁӢеәҸпјүгҖӮ

  3. CGIзЁӢеәҸе’ҢFastCGIзЁӢеәҸзҡ„еҢәеҲ«пјҡ

е…ідәҺCGIзЁӢеәҸпјҡ
CGIдҪҝеӨ–йғЁзЁӢеәҸдёҺWeb Serverд№Ӣй—ҙдәӨдә’жҲҗдёәеҸҜиғҪгҖӮCGIзЁӢеәҸиҝҗиЎҢеңЁзӢ¬з«Ӣзҡ„иҝӣзЁӢдёӯпјҢе№¶еҜ№жҜҸдёӘWebиҜ·жұӮе»әз«ӢдёҖдёӘиҝӣзЁӢпјҢиҝҷз§Қж–№жі•йқһеёёе®№жҳ“е®һзҺ°пјҢдҪҶж•ҲзҺҮеҫҲе·®пјҢйҡҫд»Ҙжү©еұ•гҖӮйқўеҜ№еӨ§йҮҸиҜ·жұӮпјҢиҝӣзЁӢзҡ„еӨ§йҮҸеҲӣе»әе’Ңй”ҖжҜҒдјҡдҪҝж“ҚдҪңзі»з»ҹжҖ§иғҪеӨ§еӨ§дёӢйҷҚгҖӮжӯӨеӨ–пјҢз”ұдәҺең°еқҖз©әй—ҙж— жі•е…ұдә«пјҢд№ҹйҷҗеҲ¶дәҶиө„жәҗйҮҚз”ЁгҖӮ

е…ідәҺFastCGIзЁӢеәҸпјҡ дёҺCGIзЁӢеәҸдёәжҜҸдёӘиҜ·жұӮеҲӣе»әдёҖдёӘж–°зҡ„иҝӣзЁӢдёҚеҗҢпјҢFastCGIдҪҝз”ЁжҢҒз»ӯзҡ„иҝӣзЁӢпјҲmasterпјүжқҘеӨ„зҗҶдёҖиҝһдёІзҡ„иҜ·жұӮгҖӮиҝҷдәӣиҝӣзЁӢз”ұFastCGIжңҚеҠЎеҷЁз®ЎзҗҶпјҢиҖҢдёҚжҳҜwebжңҚеҠЎеҷЁгҖӮеҪ“иҝӣжқҘдёҖдёӘиҜ·жұӮж—¶пјҢwebжңҚеҠЎеҷЁжҠҠзҺҜеўғеҸҳйҮҸе’ҢиҝҷдёӘйЎөйқўиҜ·жұӮйҖҡиҝҮдёҖдёӘsocketжҲ–иҖ…дёҖдёӘTCP connectionдј йҖ’з»ҷFastCGIиҝӣзЁӢгҖӮ

PHP-CGIдёҺPHP-FPM

  • PHP-CGIжҳҜPHPи„ҡжң¬и§ЈжһҗеҷЁпјҢе®ғжҳҜCGIеҚҸи®®зҡ„е®һзҺ°

  • PHP-FPMжҳҜFastCGIеҚҸи®®зҡ„е®һзҺ°

  • PHP-CGIе’ҢPHP-FPMйғҪжҳҜзЁӢеәҸ
