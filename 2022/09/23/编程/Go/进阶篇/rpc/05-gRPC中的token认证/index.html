介绍

gRPC 为每个 gRPC 方法调用提供了 token 认证支持，可以基于用户传入的 token 判断用户是否登录、以及权限…，实现 token
认证的前提是，需要定义一个结构体，并实现 grpc.PerRPCCredentials 接口。

grpc.PerRPCCredentials
